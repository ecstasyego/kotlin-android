# Design Patterns

## MVC: Model + View + Controller
- Controller : View
  - 1 : n
    - Controller : View01, View02, View03, ...

<br>

## MVP: Model + View + Presenter
- Presenter : View
  - 1 : 1
    - Presenter01 : View01
    - Presenter02 : View02
    - Presenter03 : View03
    - ...


<br>

## MVVM: Model + View + ViewModel
- ViewModel : View
  - 1 : n
    - ViewModel : View01, View02, View03, ...
